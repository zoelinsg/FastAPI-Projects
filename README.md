# FastAPI Projects

這個倉庫包含了多個基於 FastAPI 的專案範例，每個專案都展示了不同的功能和應用場景。

## 專案列表

1. **Academic Scraper**：一個學術論文爬蟲與關鍵詞分析工具。
2. **Blog**：一個簡單的部落格應用程式，允許用戶創建、閱讀、編輯和刪除部落格文章。
3. **Chat**：一個即時聊天應用程式，允許用戶即時發送和接收訊息。
4. **FAQ Chatbot**：一個問答機器人，能夠自動回應用戶的問題。

## 安裝

每個專案都有自己的安裝步驟，請參考各專案目錄中的 `README.md` 文件以獲取詳細的安裝和使用說明。

## 使用

1. 克隆此倉庫：

    ```sh
    git clone https://github.com/zoelinsg/FastAPI-Projects.git
    cd FastAPI-Projects
    ```
