# FastAPI Projects

## 專案列表
- **Academic Scraper**：一個學術論文爬蟲與關鍵詞分析工具。
- **Blog**：一個簡單的部落格應用程式，允許用戶創建、閱讀、編輯和刪除部落格文章。
- **Chat**：一個即時聊天應用程式，允許用戶即時發送和接收訊息。
- **FAQ Chatbot**：一個問答機器人，能夠自動回應用戶的問題。

## 克隆此倉庫

    git clone https://github.com/zoelinsg/FastAPI-Projects.git
